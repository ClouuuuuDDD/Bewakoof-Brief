04 / SWOT + Actionable Drivers

SWOT: Strengths In. Gaps Out.

S
Strengths
  • Youth brand equity → creator-led category drops.
  • 10M+ MAU + 70% repeat rate → loyalty flywheel ready.
  • Licensed IP (Marvel, Disney) → premium pricing spike.
  • TMRW capital → fund marketplace tooling aggressively.
  • Social-driven traffic 22% → organic cuts CAC.
W
Weaknesses
  • 85% apparel GMV → concentration risk; diversify now.
  • Return rate 8.2% (peer avg 5.5%) → fix fit guides.
  • Low AOV (₹0–25 range) → bundles + upsell needed.
  • Thin EBITDA 3.1% + 22.5% marketing spend → margin squeeze.
  • 3P logistics → inconsistent delivery, 6% repeat drop.
O
Opportunities
  • Streetwear + modest fashion boom → curated category hubs.
  • Tier 2/3: 65% new buyers → vernacular UI + fit assistant.
  • India beauty: $5B market → scale Cosmos fast.
  • Gen Z eco-aware (64%) → recycled catalog pivot.
  • Omnichannel: 100+ stores by 2027 → click-and-collect.
T
Threats
  • Meesho scaling fashion → differentiate via curation + community.
  • Digital ad costs +22% YoY → CAC pressure, ROAS drop.
  • Cotton volatility (~12% swing 2024) → margin compression.
  • Myntra: 60M MAU → youth loyalty hard to pry away.
  • Sustainability gap → Gen Z migration risk if unaddressed.
